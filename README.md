# Top MBA Colleges

This app helps users search for MBA courses and colleges based on different trades or specializations. It features a clean, modern UI with a search interface right on the home page.

Below you will find a list of user journeys, each with a brief summary and a link to the detailed guide:

1. [Search MBA Options](docs/journeys/search-mba-options.md) - Find colleges and courses using filters and keywords

For design details, see [docs/design/design-system.md](docs/design/design-system.md).

## External Services
1. Progressier - Used for enabling PWA features (offline capabilities, install prompts, etc.)
2. Sentry - Used for error monitoring and logging
3. Umami - Used for basic usage analytics
